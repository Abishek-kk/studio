# **App Name**: OnboardAI

## Core Features:

- Data Extraction: Data Interpreter Tool: Uses OCR and NLP to extract patient data from various document types, intelligently deciding when key medical information is present and relevant.
- Form Auto-Fill: Form Filler Agent: Pre-fills digital forms with extracted and verified data.
- Appointment Booker: Appointment Scheduling: Interface to pick preferred appointment times from available slots.
- Indoor Navigation: Digital Navigation: Provides turn-by-turn directions via the web app, with map integration.
- Secure Upload: Secure portal for patients to upload referral documents.
- Onboarding Status: Real-time Status Dashboard: Display of current status in onboarding process.

## Style Guidelines:

- Primary color: HSL 210, 70%, 50% (Hex: #3399E6), a vibrant blue, reflecting trust and efficiency in healthcare.
- Background color: HSL 210, 20%, 95% (Hex: #F0F8FF), a very light, desaturated blue providing a clean and calming backdrop.
- Accent color: HSL 180, 60%, 40% (Hex: #33CCCC), a contrasting cyan, is used for interactive elements.
- Font pairing: 'Inter' (sans-serif) for both headings and body text. This provides a modern, clean, and readable interface, suitable for displaying medical information.
- Use clear and universally understood icons for navigation and status indicators.
- Clean, card-based layout for easy information access.
- Subtle transitions and progress animations to guide the user through the onboarding process.
